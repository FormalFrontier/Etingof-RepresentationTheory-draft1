## Accomplished

- **Proved `quadratic_no_roots`**: A quadratic ax²+bx+c with a≠0 and non-square discriminant has no roots. Uses contrapositive: any root r gives disc = (2ar+b)², a square.
- **Proved `charW₁_elliptic`**: charW₁ = -1 on elliptic elements (0 fixed points on P¹).
  - M₀₁ ≠ 0 for elliptic g (otherwise disc = (M₀₀-M₁₁)², a square)

- **Proved `charVα₁_elliptic`**: charVα₁ = 0 on elliptic elements (no conjugate is upper triangular).

  - But disc(x⁻¹gx) = disc(g) (conjugation invariant), contradicting elliptic

  - Uses charW₁ = -1 and charVα₁ = 0 to simplify χ = (-1)·0 - 0 - Ind = -Ind
