## Accomplished

Proved `normSq_complementaryChar_parabolic` in `Lemma5_25_3.lean` (issue #1366).

The theorem states |χ(g)|² = 1 for parabolic matrices g in GL₂(F_q), where χ is the complementary series virtual character.

**Proof strategy:**
1. Proved `complementaryChar_parabolic_val`: χ(g) = -α(a) for some unit a (the repeated eigenvalue).
   - `charW₁` vanishes on parabolic (trace formula: W₁ = ε·det, parabolic has disc=0 so this contributes 0 to the virtual character difference)
   - Induced character from K vanishes on parabolic (no conjugate of a parabolic matrix lies in the elliptic subgroup K)
   - `charVα₁(g) = α(eigenvalue)` for parabolic g (class function + upper-triangular computation)

2. Key helper lemmas proved:
   - `conj_isScalar`: if x⁻¹gx is scalar then g is scalar
   - `parabolic_conj_not_in_ellipticSubgroup`: no conjugate of a parabolic matrix lies in K
   - `charVα₁_parabolic_upperTri`: charVα₁(g) = α(g₀₀) for upper-triangular parabolic g, via bijection showing |{x ∈ GL₂ : (x⁻¹gx)₁₀ = 0}| = borelCard
   - `charVα₁_parabolic`: every parabolic g has charVα₁(g) = α(a) for some unit a (triangularization via swap/shear matrices)

3. Also fixed char 2 sorry in `quadratic_one_root_zero_disc` using Frobenius surjectivity on finite fields.
