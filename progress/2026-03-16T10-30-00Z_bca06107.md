## Accomplished

Formalized statements for three core Chapter 4 character theory theorems (issue #598):

1. **Theorem 4.2.1** (`Theorem4_2_1.lean`): Characters of irreducible representations span the space of class functions. Stated as: every function `G → k` constant on conjugacy classes lies in `Submodule.span k` of characters of `Simple` FDRep objects. Uses `FDRep.character`, `CategoryTheory.Simple`.

2. **Corollary 4.2.2** (`Corollary4_2_2.lean`): The number of isomorphism classes of irreducible representations equals `Fintype.card (ConjClasses G)`. Stated existentially: there exist `n` pairwise non-isomorphic simple FDRep objects that exhaust all simples, with `n = card (ConjClasses G)`.

3. **Theorem 4.5.1** (`Theorem4_5_1.lean`): First orthogonality relation in two parts:
   - Part (i): `⅟|G| • Σ_g χ_V(g) * χ_W(g⁻¹) = finrank k (W ⟶ V)` (character inner product = dim Hom_G)
   - Part (ii): For simple V, W: inner product = 1 if V ≅ W, else 0 (matches `FDRep.char_orthonormal`)

All three files replaced `True` placeholders with proper Lean propositions. Proofs remain `sorry`. All compile cleanly (`lake build` succeeds, only expected `sorry` warnings).

Key design decisions:
- Used universe-polymorphic signatures `{k G : Type u}` to match Mathlib's `FDRep` universe constraint
- Used `Invertible (Fintype.card G : k)` (matching Mathlib's character API) rather than `IsUnit`
- Used `open scoped Classical` for `Decidable (Nonempty (V ≅ W))` in the if-then-else of 4.5.1(ii)
- Dropped unused `[DecidableEq G]` from 4.2.1 per linter
