**Mathematical analysis of the remaining sorry:**

The proof requires showing F(B₁) ≅ B₂ as B₂-modules when F is a module category equivalence and both algebras are basic. The critical missing step is **radical preservation by equivalences**: F(J₁·B₁) = J₂·F(B₁).

Proof strategy (fully worked out):
- **Easy direction** (J₂·F(B₁) ⊆ F(J₁·B₁)): F is exact, so F(B₁)/F(J₁·B₁) ≅ F(B₁/J₁). The latter is semisimple (F preserves semisimple objects). J₂ annihilates semisimple modules.
- **Hard direction** (F(J₁·B₁) ⊆ J₂·F(B₁)): For each maximal N ⊂ F(B₁), use the adjunction: Hom_{B₂}(F(J₁·B₁), F(B₁)/N) ≅ Hom_{B₁}(J₁·B₁, G(F(B₁)/N)). The latter is zero because G(F(B₁)/N) is simple (G preserves simples) and J₁ annihilates simples.

After radical preservation, the rest follows: F(B₁)/J₂·F(B₁) ≅ F(B₁/J₁) ≅ B₂/J₂ → lift by projectivity → surjective by `projective_lift_surjective` → kernel condition by splitting.
