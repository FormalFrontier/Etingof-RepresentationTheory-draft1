### Critical mathematical finding

**The convention note claiming `Φ(e_T) = |P_λ| · Σ sign(q) · δ_{toTabloid(σ_T·q)}` is FALSE.**

The claim requires `σ_T P_λ σ_T⁻¹ = P_λ` (P_λ normalized by σ_T). Counterexample: λ = (2,1), n = 3, T = [[0,2],[1]]. σ_T = (1 2). Then (0 1) ∈ P_λ but conjugate σ_T (0 1) σ_T⁻¹ = (0 2) ∉ P_λ.

Root cause: P_λ permutes within rows of the CANONICAL filling, but σ_T maps entries to positions via the SYT T, which has different row assignments.
