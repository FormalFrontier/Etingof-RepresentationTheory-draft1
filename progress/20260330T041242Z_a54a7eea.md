### Extracted and proved `sigma_contains_all_single` helper

The core irreducibility argument for `inducedRepV_simple` has been extracted into a standalone helper lemma `sigma_contains_all_single`. This proves: given a nonzero invariant subspace σ containing a function supported on a single coset q₁, σ contains ALL single-coset-supported functions at q₁.

The proof:
1. Constructs `S : Subrepresentation (FDRep.ρ U)` with carrier `{v | ∃ f ∈ σ supported on q₁ with f(q₁) = v}`
2. Proves `add_mem'`, `zero_mem'`, `smul_mem'` for the submodule
3. Proves `apply_mem_toSubmodule` via the conjugation trick: for s ∈ stabAux, acting by (1, q₁.out * s * q₁.out⁻¹) on f gives f' with f'(q₁) = ρ_U(s)(f(q₁))
4. Shows S ≠ ⊥ (contains g₁ q₁ ≠ 0)
5. Concludes S = ⊤ by simplicity → every u ∈ U has a witness in σ
