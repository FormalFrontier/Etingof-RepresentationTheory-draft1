## Accomplished

- **Fixed critical bug in RelColumnSubgroup**: The original definition used `MulAut.conj σ_T` (giving σ_T Q_λ σ_T⁻¹) but the correct James column stabilizer is C_T = σ_T⁻¹ Q_λ σ_T, requiring `MulAut.conj (sytPerm n la T)⁻¹`
- **Proved P_λ ∩ Q_λ = {1}** (`RowSubgroup_inter_ColumnSubgroup`): A permutation preserving both rows and columns must be the identity, since each cell is uniquely determined by (row, col)
- **Proved ColumnSubgroup_ne_tabloid**: Non-identity column permutations produce different tabloids (eliminates 1 sorry)
- **Proved RelColumnSubgroup_ne_tabloid**: Entry-level column permutations change the tabloid (eliminates 1 sorry), derived from the above
- **Added dominance order infrastructure**: `tabloidCumulCount`, `tabloidDominates`, `tabloidStrictDominates` with reflexivity, transitivity, well-definedness on quotient
- **Added last-letter total order infrastructure**: `tabloidRowVec` with well-definedness and bidirectional equivalence with tabloid equality
