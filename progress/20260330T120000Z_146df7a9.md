4. **`mulLeft_youngSym_zero_of_ne`**: Off-diagonal case — construct A-linear map via right multiplication, apply Schur's lemma + `Theorem5_12_2_distinct`
5. **`trace_mulLeft_youngSym_eq`**: Diagonal case — factor T = ι∘π through ℂ using sandwich property (Lemma5_13_1), apply `trace_comp_comm'`, compute `π∘ι = α·id`

### Key techniques

- **Base change through ℤ**: Both ℚ and ℂ Young symmetrizers come from ℤ via `mapRangeRingHom`. Injectivity of ℤ → ℚ lifts the ℚ identity to ℤ, then maps to ℂ.
- **Sandwich property**: `c_λ * x * c_λ` is proportional to `c_λ` (derived from Lemma5_13_1). Combined with `c(1) = 1`, this gives a clean factorization T = ι∘π.
- **Schur's lemma for off-diagonal**: Non-isomorphism of distinct Specht modules means any A-linear map between them is zero.
