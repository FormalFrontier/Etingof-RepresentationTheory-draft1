The proof framework:
1. **Row absorption lemma** (`of_row_mul_youngSymmetrizer'`): of(p) · c_λ = c_λ for p ∈ P_λ, proved from `of_row_mul_RowSymmetrizer`
2. **Column inversion count** (`columnInvCount'`): counts pairs in same column with inverted order under σ
3. **Column-standard predicate** (`isColumnStandard'`): all columns increasing under σ
4. **Base case** (`column_standard_in_span'`): column-standard σ maps to SYT polytabloid via row-sorting within right P_λ-coset, using row absorption
5. **Inductive step**: Garnir reduction expresses non-column-standard σ · c_λ as combination with fewer column inversions
6. **Main theorem**: well-founded induction combining base and inductive cases — PROVED (modulo sorry'd helpers)
