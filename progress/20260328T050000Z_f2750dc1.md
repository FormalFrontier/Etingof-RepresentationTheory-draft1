## Accomplished

Proved the fullness half of `exists_full_idempotent_basic_corner` in `BasicAlgebraExistence.lean`. This was the main sorry in issue #1848.

Key results proved:
1. **`pi_matrix_single_generates_ideal`**: In a product of matrix algebras ∏ Mat_{n_i}(k), the sum of matrix units E₁₁ generates the whole ring as a two-sided ideal. Required careful handling of dependent Pi types (Pi.single for dependent families).

2. **Fullness of e = ∑ e_lifted i**: Proved `Ideal.span {a * e * b | a b} = ⊤` via:
   - Transport `pi_matrix_single_generates_ideal` through Wedderburn-Artin isomorphism φ to A/J(A)
   - Show Ideal.map of AeA under quotient map is ⊤
   - Deduce AeA + J = A via `Ideal.mem_quotient_iff_mem_sup`
   - Use `IsNilpotent.isUnit_one_sub` (works for non-commutative Ring, unlike `isUnit_of_sub_one_mem_jacobson_bot` which needs CommRing)
   - Conclude AeA = ⊤ via `Ideal.eq_top_of_isUnit_mem`

3. **Technical challenges overcome**:
   - `Pi.single` for dependent Pi types: simp lemmas don't work directly; used `Function.update` + `dite_apply` + `split/subst` pattern
   - Non-commutative ring: `linarith`/`linear_combination` need CommSemiring; used manual algebra
   - Import: added `Mathlib.RingTheory.Jacobson.Ideal` for Jacobson radical theory
