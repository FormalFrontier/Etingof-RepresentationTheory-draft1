Worked on issue #700: prove `character_div_dim_isIntegral` (Theorem 5.4.4 helper).

- **Proved character is algebraic integer**: `V.character g` is a sum of roots of unity (from `character_eigenvalue_decomposition`), each satisfying `X^m - 1 = 0`, hence integral over ℤ. Used `IsIntegral.sum` and `Polynomial.monic_X_pow_sub_C`.

- **Proved Bezout reduction**: Using `Nat.Coprime` and `Nat.gcd_eq_gcd_ab`, extracted integer coefficients `a, b` with `a·|C| + b·dim(V) = 1`. Then showed `χ(g)/dim(V) = a·(|C|·χ(g)/dim(V)) + b·χ(g)`, reducing integrality to two ingredients: character integrality (proved) and class-sum-scalar integrality (Prop 5.3.2).

- **Blocked on Prop 5.3.2**: The class-sum-scalar integrality `IsIntegral ℤ (|C| * χ(g) / dim(V))` requires:
  1. Constructing the class sum `∑_{h ∈ C(g)} ρ(h)` as a G-equivariant endomorphism (Action.Hom in FDRep)
  2. Applying Schur's lemma (`endomorphism_simple_eq_smul_id`) to get scalar · id
  3. Extracting the scalar via trace
  4. Showing integrality via `IsIntegral.sum` + `isIntegral_algebraMap_iff`

  The equivariance proof and Schur application work at the categorical level, but extracting the linear map equation from the categorical equation requires navigating 3 levels of `.hom` unwrapping (Action.Hom → FGModuleCat morphism → ModuleCat morphism → LinearMap). This plumbing is the blocker.
