## Current frontier

The only remaining sorry is `spechtCharacter_eq_alternating_sum_permCharacter` (line ~249),
which is the deep representation-theoretic core of the Frobenius character formula. This
theorem states that the alternating sum of shifted power sum coefficients equals the
Specht module character.

**Blocked on Young's Rule**: The proof requires the character decomposition
`χ_{U_μ}(σ) = Σ_ν K_{ν,μ} · χ_{V_ν}(σ)` (Young's Rule), which is NOT formalized.

The infrastructure needed:
1. **Semisimple decomposition** → multiplicities (Mathlib has `IsSemisimpleModule k[G] V`)
2. **Schur's lemma** → dim Hom = multiplicity (Mathlib has this)
3. **Prop 5.14.1** → Hom(U_λ, V_μ) dimensions (proved in codebase)
4. **Bridge**: Connect dim Hom to Kostka numbers (Definition5_14_2 defines them)
5. **Alternating sum identity**: Σ_π sign(π) K_{ν,sort(λ+ρ-π(ρ))} = δ_{ν,λ}
