### Analysis of the column identity proof

The column identity ∑_u w(u,c) = HP(μ)/HP(μ\c) is the hardest step in the GNW hook walk proof. Multiple proof strategies were analyzed:

- **Strong induction on |μ|**: Removing a corner c₀ ≠ c changes hook walk weights in a cascading way (the hook of every cell in c₀'s arm/leg changes, which propagates through the recursive definition). No clean multiplicative relationship between w_μ and w_{μ\c₀} exists.
- **Direct algebraic manipulation**: All attempts to derive the column sum from the recursion produce tautologies (S = S). The column identity is "external" — it can't be derived from the recursion + row sum property alone.
- **Fubini/double counting**: Only works if we already have the column identity.
- **Bypassing the column identity**: The hook quotient identity ∑_c HP/HP_c = |μ| IS the inductive step for the hook length formula. Proving it directly requires the same level of difficulty.

The key missing infrastructure is a **relationship between w_μ(u, c) and w_{μ'}(u, c)** when μ' = μ\c₀ for some corner c₀. This would enable the inductive step.
