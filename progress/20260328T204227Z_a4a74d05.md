**Proved `sum_youngSym_permTracePoly_eq_alpha_schurPoly`** — the Frobenius + orthogonality step of the Weyl character formula. This reduces sorry 2 (the combinatorial identity ∑_σ c_λ(σ)·permTracePoly(N,σ) = α·s_λ) to a single character orthogonality lemma.

The proof uses the **antisymmetric basis argument**:
1. Multiply both sides by the Vandermonde determinant Δ
2. Show `F*Δ - α·A_{λ+δ} = 0` using `antisym_eq_zero`
3. Antisymmetry: F is symmetric (sum of power sum products), Δ is antisymmetric
4. Alternant coefficients: for BPs, use `youngSym_charValue_orthogonality`; for non-BPs, use homogeneity + `exists_bp_of_strictAnti_sum`
