### Proved: `not_hasHomologicalDimensionLE_zero_polynomial`
- For any nontrivial commutative ring R, `Polynomial R` has global dimension ≥ 1
- Uses `Module.AEval' (0 : R →ₗ[R] R)` to construct the augmentation module (R with X acting as 0)
- Key argument: any R[X]-linear section of the surjection R[X] → R is killed by X (since X acts as 0 on R), but multiplication by X is injective in R[X], so the section is zero, contradicting surjectivity

### Proved: `Polynomial.X_mul_eq_zero`
- Multiplication by X is injective in R[X] (for any ring R), using coefficient comparison
