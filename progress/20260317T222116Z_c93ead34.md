### Proved
- `Etingof.posdef_min_value`: For a Dynkin diagram, B(x,x) >= 2 for all nonzero x in Z^n. Uses positive definiteness (B(x,x) > 0) combined with evenness (Lemma 6.4.2) to conclude B(x,x) >= 2.
- `Etingof.Theorem_6_5_2a_finiteness`: Structure proved — reduces finiteness of positive roots to a coordinate bound via `Set.Finite.subset (Set.finite_Icc 0 n)`.

### Remaining sorry
- `Etingof.posRoot_coord_le`: Each coordinate of a positive root d satisfies d_i <= n. This is true for all Dynkin types (verified: A_n max=1, D_n max=2, E_6 max=3, E_7 max=4, E_8 max=6, all <= n) but proving it requires either:
  - Eigenvalue analysis of the Cartan matrix (casting to Q or R)
  - Tree-structural induction using the coupled inequalities d_i <= (1 + sum_adj d_j)/2
  - Both approaches require significant infrastructure not currently available

### Key insight discovered
For positive roots with multiple nonzero coordinates, the bilinear expansion B(d-e_i, d-e_i) >= 2 yields (Ad)_i <= 1, i.e., 2d_i - sum(adj d_j) <= 1. This bounds each coordinate in terms of its neighbors, but deriving an absolute bound requires traversing the tree structure, which is hard with a matrix-based graph representation.
