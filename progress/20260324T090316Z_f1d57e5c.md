### Proved hσ_surj (surjectivity of block assignment σ) — issue #1669

Eliminated the last sorry in `Theorem9_2_1.lean` by proving `Function.Surjective σ` where `σ : ι → Fin n` maps simple module classes to Wedderburn-Artin blocks.

**Proof strategy**: By contradiction. If `l₀ ∉ range σ`:
1. The rank property gives `smulRange (M j) (e_raw ⟨l₀, 0⟩) = ⊥` for all j (since `σ j ≠ l₀` for all j)
2. So `e_raw ⟨l₀, 0⟩` annihilates every simple module M j
3. For any maximal left ideal I, the quotient A/I is simple, hence isomorphic to some M j₀ by `hM_exhaustive`; transporting annihilation through the isomorphism shows `e_raw ⟨l₀, 0⟩ ∈ I`
4. Since this holds for all maximal left ideals, `e_raw ⟨l₀, 0⟩ ∈ Ring.jacobson A`, so `π(e_raw ⟨l₀, 0⟩) = 0`
5. But `π(e_raw ⟨l₀, 0⟩) = WA.symm(Pi.single l₀ (E₁₁)) ≠ 0` — contradiction

**Key insight**: The link between module annihilation and Jacobson radical membership via maximal left ideals + exhaustiveness avoids needing to construct an A/J-module structure on the simples.
