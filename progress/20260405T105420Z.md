## Current frontier

The single sorry in `exists_surjection_with_trivial_kernel_head` (line ~499) remains. The proof requires:

1. **Surjection construction** (hardest part): Need to show ∃ surjection g : F(B₁) → B₂/J₂. This requires showing that the heads F(B₁)/J₂·F(B₁) and B₂/J₂ are isomorphic semisimple modules. The key computation (dim_k Hom(F(B₁), S) = 1 for each simple S) is mathematically proven but formalizing the "combine surjections to individual simples into a surjection to the head" step needs infrastructure for semisimple module decomposition not yet in Mathlib.

2. **Kernel condition**: ker f ≤ J₂ · ker f. Follows from splitting (B₂ projective) + head dimension matching, once the surjection is constructed.

**Missing Mathlib infrastructure** (any one suffices):
- Radical preservation by equivalences (~150 lines, see #2138)
- Krull-Schmidt theorem for modules over Artinian rings
- Projective cover existence/uniqueness for semiprimary rings
- Semisimple module isomorphism criterion (two semisimple modules with same simple multiplicities are isomorphic)

## Next step

The most promising approach to close the sorry is to prove **semisimple module isomorphism by multiplicities**: two finite semisimple modules over B₂/J₂ ≅ k^n with each simple appearing once are isomorphic. This would allow constructing the surjection F(B₁) → B₂/J₂ via the head isomorphism. Estimated difficulty: 7, ~120-150 lines.

Alternatively, prove radical preservation (#2138) which gives F(J₁·B₁) = J₂·F(B₁), reducing the head isomorphism to F's exactness.

## Blockers

Missing Mathlib infrastructure for any of: Krull-Schmidt, projective covers, or semisimple module decomposition/classification. The proof requires at least one of these results, which would need to be built from scratch (~100-200 lines each).
