### Proof architecture insight

The circular dependency that blocked this issue:
- `sum_youngSym_permTracePoly_eq_alpha_schurPoly` uses `youngSym_charValue_orthogonality`
- To prove `youngSym_charValue_orthogonality` from polynomials requires `sum_youngSym_permTracePoly_eq_alpha_schurPoly`

The solution: break the cycle via representation theory. The trace Kronecker identity (`youngSym_trace_kronecker`) is proved independently using Specht module theory (Schur's lemma + sandwich property), bypassing the polynomial world entirely.
