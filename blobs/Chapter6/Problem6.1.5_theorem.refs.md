# References: Theorem: Connected quiver is finite type iff Dynkin

## External Dependencies

- **Gabriel's theorem: a connected quiver has finitely many indecomposable representations if and only if its underlying graph is a Dynkin diagram of type ADE** (external_result)
  Gabriel's theorem is NOT in Mathlib. Quiver representations have basic support (`Quiver`, `Representation`), but Gabriel's classification is absent. This is a major theorem that would need full formalization.
