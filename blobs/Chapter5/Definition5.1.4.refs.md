# References: Frobenius-Schur indicator

## Mathlib Coverage (gap)

The Frobenius-Schur indicator (0 for complex type, 1 for real, -1 for quaternionic) is not in Mathlib. Needs to be defined from scratch.

## External Sources (definition gap)

  Definition and computation of ν(χ) = (1/|G|) Σ χ(g²); characterization as 0, 1, or -1
- **[other_formal]** MathComp (Coq) — character.v, cfun_indicator
  Frobenius-Schur indicator formalized in Coq's character theory library
